## 개요

### 서비스 명: Every Minton

### 목적
배드민턴 모임에서 운동 일정을 진행할 때, 모임의 운영진들이 운동에 참여한 모임원들의 시간당 게임 횟수가 최대한 비슷하게 유지되도록 게임(매칭)을 만들어 일정이 원활하게 진행되도록 도와줌.
**시간당 게임 횟수 = 진행한 게임 횟수 / 참여한 일정에 참여한 시간(분)  * 60** 

### 제공되는 기능
일정을 생성하면 해당 일정에 참여하는 모임원들의 게임을 

#### 사용자에게 보여질 정보 (어떻게 보여질지는 미정)
- 현재 참여중인 모임원들의 시간당 게임 횟수
- 현재 참여중인 모임원들의 대기시간 (현재시간 - 마지막 게임 종료 시간)
- 현재 참여중인 모임원이 진행한 게임 수
- 모임원의 이름, 성별, 급수(실력)

#### 그 외 기능
- 일정 생성/관리 
- 한번 등록된 유저는 서버에 저장되어 추후 일정에 참여 인원 추가 시 이름만 입력하여 추가
- 수동으로 게임 생성
- 자동으로 게임 생성
